## 使用言語

```sh
rubyonrailsのAPIを使用してReactを使用しています
Reactのライブラリはtodo_api_userというフォルダの中にあります。
```

## 使用技術

```sh
railsAPIサーバーの作成
react axiosによる非同期処理によるAPIのデータの取得、送信
```

## 使用 GUI ツール

```sh
API postman
DB mysqlworkbench
```

## 立ち上げ

```sh
rails rails s -p 3001
react    npm start
```
